Modulo

  • Only can be used by int operands
    • 5 % 4 = 1
    • 7 % 88 = 7
    • -20 % 7 = -6
    • 20 % -7 = 6
    • -20 % -7 = -6

Mean of 3 Numbers


#include <stdio.h>
int main(void){
  float num1, num2, num3, sum, average;
  printf("Enter 3 number: \n");
  scanf("%f",&num1);
  scanf("%f",&num2);
  scanf("%f",&num3);
  sum = num1 + num2 + num3;
  average = sum / 3;
  printf("mean = ");
  printf("%f\n", average);
  return 0;
}

General rules of type conversion

  • If either operand is long double, convert the other to long double.
  • Otherwise, if either operand is double, convert the other to double.
  • Otherwise, if either operand is float, convert the other to float.
  • Otherwise, convert char and short to int.
  • Then, if either operand is long, convert the other to long.

Effect of types

  • Type of operands determines the type of the result
    • The type of output is the type of operands (after conversion)
  • int <op> int → int
  • int <op> long → long
  • float <op> float → float
  • float <op> int → float
  • double <op> float → double

Effect of types

  • If both operand of division (/) is int
  • → data lost
