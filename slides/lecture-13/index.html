File postfix

  • Most compilers consider the source code file postfix
    • .c → C code
    • .cc .cpp → C++ code

A program in multiple file

  • We can create our .h files
  • These files contains structs, functions declaration, etc.
  • func.c
    • 
#include <stdio.h>

void f(int x) {
  printf("%d", x);
}
  • func.h
    • 
#ifndef FUNC_H
#define FUNC_H
void f(int);
#endif
  • Then includes header to use
  • main.c
    • 
#include "func.h"

int main(void) {
  f(20);
}

Preprocessor Command

  • We can use preprocessor commands to control how our code is compiled and etc.
    • Conditional compilation
  • Main preprocessor commands
    • 
#define XYZ // define XYZ as a preprocessor definition (value is not important)
#ifdef XYZ  // is true if XYZ is defined
#ifndef XYZ // is true if XYZ is not defined
#if XYZ     //  is true if XYZ != 0
#endif      // End of a if block
    

#include <stdio.h>

#define ABC
#define XYZ 1
int main(){
  #ifdef ABC
  printf("ABC is defined \n");
  #endif
  printf ("I am here\n");
  #if XYZ
  printf("XYZ is defined and is not 0\n");
  #endif
}
