Decision

  • Decisions are based on conditions
    • If it is snowing then We will cancel the game
    • If the class is not canceled then I will attend else I will go to gym
  • In programming
    • Do statements based on conditions
      • True: The statements will be done
      • False: The statement wont be done

Conditions

  • Conditions by comparisons; e.g.,
    • Weather vs. snowing
    • Variable x vs. a value
  • Comparing numbers: Relational Operators

Even-Odd


#include <stdio.h>
int main(void){
  int number_to_test, remainder;
  printf("Enter your number to be tested: ");
  scanf("%d", &number_to_test);
  remainder = number_to_test % 2;
  if(remainder == 0)
    printf ("The number is even.\n");
  else
    printf ("The number is odd.\n");
  return 0;
}

    • <Statements 2> becomes an if-else statement
